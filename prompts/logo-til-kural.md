# Промпт: разработка лого til-kural

Готовый промпт под til-kural — два варианта: короткий для AI-генератора (Midjourney / Imagen / SDXL / DALL·E) и развёрнутый бриф для дизайнера. Бренд-вводные взяты из текущего кода (палитра, отсылки к Байтұрсынұлы, домен в Сатпаеве).

---

## 1. Короткий промпт для AI (English, MJ-стиль)

```
Logo for "Тіл-құрал" — Kazakh-language learning center in Kazakhstan.
Concept: a single elegant mark uniting a quill/brush stroke, an open book,
and the Kazakh letter "Т" — referencing Akhmet Baitursynuly's alphabet
reform legacy, the cultural roots, and modern AI-assisted education.

Style: minimal vector mark, geometric, confident, slightly Bauhaus + subtle
Kazakh ornament cue (oyu-örnek), no kitsch. Bold negative space. Works mono.

Palette: deep navy #0F1B2D, warm gold #F5C518, off-white #F8EFE0.
Primary mark in gold on navy, plus a one-color black version.

Output: clean lockup with wordmark "Тіл-құрал" set in a strong humanist
serif (or grotesque with a wide kazakh-friendly cyrillic) underneath the
icon, generous letter-spacing, optical centering.

Avoid: stock book/graduation hat clichés, photoreal textures, gradients,
emoji, AI-generated typography, cyrillic letters with broken kerning,
yurt/eagle stereotypes, heavy national ornament borders.

--ar 1:1 --style raw --v 6
```

(в Imagen / DALL·E `--ar` и `--v` убери — они не парсятся.)

## 2. Расширенный бриф (для живого дизайнера или Claude/Gemini в роли арт-дира)

**Заказчик:** КГУ «УМЦ Тіл-құрал», Сатпаев, БИН 241240033540. Единственный профильный центр обучения казахскому языку в Ұлытауской области.

**Что такое продукт:** AI-платформа изучения казахского языка (тесты, диалог, фото-проверка рукописей, КАЗТЕСТ, геймификация). Сайт til-kural.kz.

**Ассоциации, которые нужно вшить:**
- Тіл-құрал = «языковой инструмент» (буквальный перевод).
- Наследие — Ахмет Байтұрсынұлы, реформа казахской письменности.
- Сатпаев / Ұлытау — корни и горно-индустриальная идентичность региона (без штампа «горы и юрта»).
- AI / современность — но не «шестерёнки и нейросети», это второй план.

**Что НЕ должно быть:** диплом со свитком, выпускной колпак, открытая книга-стоковая, флаг, орёл, шанырак, надписи стилизованные «под арабицу», эмодзи-перо.

**Палитра:**
- `tk-night` #0F1B2D (основной фон)
- `tk-gold` #F5C518 (акцент / иконка)
- `tk-blue-dark` (сине-стальной)
- `tk-beige` #F8EFE0 (вторичный фон)
- Логотип должен жить в гольд-на-нейви, на белом и в чёрном моно.

**Форм-фактор:** vector-only (SVG). Минимум 3 версии:
1. Знак (icon, квадрат 1:1, для favicon/avatar).
2. Горизонтальный лок-ап: знак + «Тіл-құрал».
3. Вертикальный: знак сверху, под ним «Тіл-құрал», а ниже мелким — «Оқыту-әдістемелік орталығы / УМЦ».

**Идеи для знака (выбрать одну, не миксовать):**
- (a) **Перо-стрела вверх** — стилизованный штрих пера, развить в «след» = строка письма.
- (b) **Лигатура «Т·қ»** — авторская, на основе казахского `қ` с диакритикой как акцент-метка.
- (c) **Раскрытая страница, образующая «Т»** — отрицательное пространство = буква.
- (d) **Стилизованный «құрал» (инструмент)** — циркуль/линейка-перо как метафора «языкового инструмента».

**Типографика:** для кириллицы и казахских доп-букв (ә, ғ, қ, ң, ө, ұ, ү, һ, і) шрифт обязан иметь полный казахский набор. Рекомендации: Manrope, Inter Display, Onest, Fact Kazakh, или платный Druk Cyrillic для жирного варианта.

**Тестовые сценарии для приёмки:**
- Favicon 16×16 — мономатичен и читается.
- Печать чёрно-белая на бланке.
- Аватар Telegram/WhatsApp (круглая маска).
- На белой и на тёмной шапке.
- Рядом с логотипом ЖезУ/госструктур — не выбивается.

**Дедлайн / итерации:** 3 концепта на ч/б эскизе → 1 на доработку в цвете → финал в SVG + PNG (256/512/1024) + brand guide одностраничный.

## Как переиспользовать для другого проекта

1. Заменить контекст:
   - название и расшифровку,
   - культурные/исторические якоря (для Solana-проекта это будет network speed / DeFi / NFT-эстетика),
   - ассоциации, которые надо вшить и которые избегать.
2. Палитру — взять из существующего бренда (CSS-переменные, Tailwind tokens) или из мудборда.
3. Сценарии приёмки оставить почти как есть — favicon, аватар, ч/б, рядом с партнёрами.
