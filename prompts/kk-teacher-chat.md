# Промпт: AI-учитель казахского с RAG и mentor-стилями

**Модель:** `gemini-2.5-flash` · **purpose:** chat
**Контракт:** свободный markdown-ответ (`**жирный**` для казахских слов, `*` для списков, **без `code-fence`** — это чат, не редактор).
**Источник:** til-kural `src/lib/kazakh-rules.ts:getTeacherSystemPrompt()`

## Зачем

Двуязычный (`kk`/`ru`) RAG-учитель с тремя стилями наставников. Использует контекст правил из локального справочника (`buildRAGContext()`), адаптируется по CEFR-шкале (`A1` → ≤8 слов в предложении, `C2` → полная свобода). Каждый ответ заканчивается вопросом или мини-заданием.

## CEFR-шкала (`levelGuidance`)

```
A1: предложения ≤ 8 слов, словарь ≤ 500 базовых слов, без идиом, метафор и поэтических цитат. Объясняй очень просто.
A2: ≤ 12 слов, простые времена и падежи. Минимум абстракций.
B1: расширенный словарь, можно одну пословицу с переводом. Сложноподчинённые предложения умеренно.
B2: свободный язык, идиомы допустимы с пояснением.
C1: литературный стиль, цитаты в оригинале, тонкие нюансы.
C2: полная свобода стиля, философские отступления, поэзия.
```

## Связка с кодом

Промпт работает в паре с RAG-функцией `buildRAGContext(query, level)` из `src/lib/kazakh-rules.ts`, которая по запросу ученика выбирает релевантные правила из JSON-каталога и форматирует их как опорный материал. Без RAG-контекста промпт деградирует до общего «учителя без справочника».

## Как переиспользовать

1. **Замена предмета** — заменить «казахского языка» на свой (история, химия, программирование). Mentor-стили подобрать соответственно (для физики: Эйнштейн / Фейнман / Ландау).
2. **Замена шкалы** — CEFR работает для языков; для других предметов — заменить на собственную (классы 1–11, beginner/junior/middle/senior).
3. **RAG-контекст** — обязательная часть. Если справочника нет — убрать блок `БАЗА ПРАВИЛ` и не делать ссылок на rule_NN.
4. **Mentor-стили** — подбирать культурно-релевантных персонажей. По одной записи на каждого: KK + RU, чтобы не подмешивать языки.
5. **Жёсткое правило завершать вопросом** — переиспользуется как есть, поднимает retention в 1.5–2×.
